# imginfo

Using exteact files containing information about the image .

# Requirements:

Pillow library installaion :

```
python -m pip install Pillow
```

# Installation: 

```
git clone https://github.com/laser010/imginfo
cd imginfo
python imginfo.py -h
```

# Examples

```
python imginfo.py -i test.jpg
```
